OAuth 2.0主要角色

OAuth 令牌类型

  • 访问令牌(Access Token)

用于代表一个用户或服务直接去访问受保护的资源

  • 刷新令牌(Refresh Token)

用于去授权服务器获取一个新的访问令牌

  • 授权码(Authorization Code Token)

仅用于授权码授权类型，用于交换获取访问令牌和刷新令牌

  • Bearer Token

不管谁拿到Token都可以访问资源，像现钞

  • Proof of Possession(PoP) Token

可以校验client是否对Token有明确的拥有权
